El error por sesgo (bias) es la diferencia entre la predicciones esperadas (promedio) y el valor correcto (punto dado).
El error por varianza es la variación entre cada una de las distintas predicciones que hace un modelo para un punto dado.
No se puede reducir a cero el error por sesgo y varianza al mismo tiempo. La disminución de uno implica el incremento del otro.
